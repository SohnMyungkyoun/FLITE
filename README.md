# 📸 Flite - 비파괴 사진 편집기

라이트룸 스타일의 폴더 기반 비파괴 사진 편집 애플리케이션입니다.  
원본 파일을 건드리지 않고 편집 정보만 별도로 저장하여 관리합니다.

![Flite Screenshot](https://img.shields.io/badge/Status-Beta-yellow)
![Platform](https://img.shields.io/badge/Platform-Web-blue)

## ✨ 주요 기능

### 🗂️ 폴더 기반 워크플로우
- **VSCode 스타일 인터페이스**: 폴더를 열어 모든 사진을 한눈에 확인
- **사이드바**: 파일 목록으로 빠른 탐색
- **썸네일 스트립**: 하단 필름스트립으로 빠른 사진 전환
- **키보드 네비게이션**: 화살표 키로 이미지 이동

### 🎨 전문적인 편집 도구
| 도구 | 설명 | 범위 |
|------|------|------|
| **노출** | 전체적인 밝기 조정 | -100 ~ +100 |
| **대비** | 명암 차이 조정 | -100 ~ +100 |
| **밝은 영역** | 하이라이트 복구 | -100 ~ +100 |
| **어두운 영역** | 그림자 복구 | -100 ~ +100 |
| **흰색** | 가장 밝은 영역 미세 조정 | -100 ~ +100 |
| **검정** | 가장 어두운 영역 미세 조정 | -100 ~ +100 |

### 💾 비파괴 편집 시스템
- ✅ **원본 보호**: 원본 파일은 절대 수정되지 않음
- ✅ **편집 정보 저장**: LocalStorage에 JSON 형태로 저장
- ✅ **실시간 적용**: 저장된 편집은 자동으로 불러와짐
- ✅ **시각적 표시**: 편집된 사진에 녹색 점(●) 표시

### 🚀 기타 기능
- 🔄 실시간 미리보기
- 📤 편집된 사진 내보내기 (PNG)
- 💾 편집 내용 저장/불러오기
- 🔄 편집 초기화
- ⌨️ 키보드 단축키

## 🚀 시작하기

### 1. 실행

**로컬 서버로 실행 (필수)**

```bash
# flite 폴더로 이동
cd ~/Documents/SNU25/Humans\ of\ SNU/flite

# Python 서버 실행
python3 -m http.server 8000

# 브라우저에서 접속
# http://localhost:8000
```

> ⚠️ **중요**: 폴더 선택 기능을 사용하려면 반드시 로컬 서버로 실행해야 합니다.

### 2. 워크플로우

#### ① 폴더 열기
1. "📁 폴더 열기" 버튼 클릭
2. 사진이 들어있는 폴더 선택
3. 모든 이미지가 자동으로 로드됨

#### ② 사진 탐색
- **사이드바**: 파일 목록에서 클릭
- **썸네일 스트립**: 하단 썸네일 클릭
- **키보드**: ← → 화살표로 이동
- **Home/End**: 첫/마지막 사진으로 이동

#### ③ 편집하기
1. 오른쪽 패널의 슬라이더로 조정
2. 실시간으로 미리보기 확인
3. "💾 편집 저장" 클릭하여 편집 정보 저장
4. 편집된 사진은 녹색 점(●)으로 표시됨

#### ④ 내보내기
- "📤 내보내기" 버튼으로 편집된 사진을 새 파일로 저장
- 원본은 그대로 유지됨
- 파일명: `원본이름_edited.png`

## 💡 팁

### 편집 순서 (권장)
1. **노출** 조정으로 전체 밝기 설정
2. **대비**로 선명도 조정
3. **밝은/어두운 영역**으로 세부 조정
4. **흰색/검정**으로 미세 조정

### 비파괴 편집의 장점
- 원본은 항상 안전하게 보관됨
- 언제든지 편집 내용 수정 가능
- 여러 버전 테스트 후 마음에 드는 것만 내보내기
- 편집 정보는 LocalStorage에 자동 저장

### ⌨️ 키보드 단축키
| 키 | 동작 |
|---|---|
| `←` `→` | 이전/다음 사진 |
| `Home` | 첫 번째 사진 |
| `End` | 마지막 사진 |

## 📦 기술 스택

- **HTML5** - 구조
- **CSS3** - 다크 테마 스타일링
- **JavaScript (Vanilla)** - Canvas API 이미지 처리
- **LocalStorage** - 편집 데이터 저장
- **File System Access API** - 폴더 열기

### 데이터 구조

```javascript
// LocalStorage
{
  "flite_edits": {
    "photo1.jpg": {
      "exposure": 15,
      "contrast": -10,
      "highlights": -20,
      "shadows": 30,
      "whites": 5,
      "blacks": -5
    },
    "photo2.jpg": { ... }
  }
}
```
